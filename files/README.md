# Pacote Google Chrome (.deb)

Coloque aqui o arquivo oficial:

`google-chrome-stable_current_amd64.deb`

Baixe a partir do site do Google Chrome para Linux (deb, 64 bits) e renomeie ou mantenha o nome esperado pelo playbook `kiosk_setup.yml`.

Sem este arquivo, a instalação do Chrome falha de propósito (política: somente via .deb).
